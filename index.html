FC'26 Rump Session NFT

Unofficial paper author NFTs for FC 2026. Prove email authorship via ZK-TLS to claim yours.

How this works: Each FC'26 rump session paper has an NFT that authors can claim by proving they control an email address listed on the paper. The claim flow uses ZK-TLS (Sigstore attestation + ZK proofs) running entirely in GitHub Actions — no trusted server required.
  1. Select your paper and enter your email — your email is checked locally against a hash and never leaves your browser
  2. Open a GitHub issue containing only your email's hash (not the email itself)
  3. A GitHub Action sends a verification code to your email
  4. Reply with the code — the action generates a ZK proof and mints your NFT on Base Sepolia
